INTRODUÇÃO

O processo de agendamento manual de espaços acadêmicos, como laboratórios e auditórios, apresenta falhas de organização e baixo controle. O projeto AgendDev propõe uma plataforma web para otimizar esse processo, oferecendo praticidade, transparência e melhor comunicação entre professores e gestores.
  Problema: A ausência de um sistema automatizado gera sobreposição de horários, falta de controle institucional e dificuldades na gestão dos espaços acadêmicos.
  Objetivo: Desenvolver uma plataforma web que centralize os agendamentos de laboratórios e auditórios, garantindo eficiência, acessibilidade e integração entre docentes e administração.


METODOLOGIA

O desenvolvimento do AgendDev seguiu as etapas de:
  Levantamento de Requisitos: Identificação das necessidades dos usuários para definir funcionalidades essenciais do sistema, como integração com o calendário acadêmico e controle de acessos.
  Desenvolvimento: Implementação da plataforma utilizando tecnologias web modernas, com foco em usabilidade e integração com banco de dados relacional.
  Testes e Validação: Testes funcionais para garantir a operação adequada de todas as funcionalidades.
  Implantação e Monitoramento: Implementação do sistema e monitoramento contínuo para melhorias.


REVISÃO BIBLIOGRÁFICA

A automatização de processos de agendamento acadêmico é amplamente reconhecida por aumentar a eficiência e reduzir falhas administrativas. De acordo com Atkinson (2018), sistemas online de agendamento contribuem para maior transparência e melhor comunicação institucional.
  Referência: ATKINSON, S. (2018). Lessons from a Pilot Program Using Google Calendar.
  Disponível em: [https://crl.acrl.org/index.php/crl/article/view/16888](https://crl.acrl.org/index.php/crl/article/view/16888)


RESULTADOS ESPERADOS

O AgendDev visa oferecer maior visibilidade e controle dos espaços acadêmicos, reduzindo conflitos e otimizando o uso dos laboratórios e auditórios. A plataforma amplia a acessibilidade digital e a inclusão de docentes, fortalecendo a comunicação e podendo ser expandida para outros ambientes. Assim, consolida-se como uma solução moderna e eficiente, voltada à inovação e à transformação digital na gestão acadêmica.
